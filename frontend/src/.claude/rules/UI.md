# 🍏 UI/UX Standards
- **Aesthetic:** Apple-inspired, glassmorphic, elegant.
- **Naming:** `className="[ComponentName] [tailwind-classes]"` is mandatory.
- **Transitions:** Every example in the SPA view must use a unique GSAP entrance/exit (Stagger, Fade-Slide, Scale-Up).
- **Accessibility:** React Aria for all Modals, Tabs, and Tooltips.
