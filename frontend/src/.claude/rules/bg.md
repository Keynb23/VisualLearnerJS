# 🌌 Background Engine
1. **SVG Path:** Morphing vector shapes (Lightest).
2. **Canvas:** Interacting particles (Medium).
3. **R3F Gradient:** 3D noise/mesh (Heavy).
4. **Gif Mask:** Transparent GIF utilized as a mask layer over a solid OKLCH color.

## Performance
- Global `performance_mode` state (Speed/Off/On) controls all 4 modes.
