# 🗺️ Project Map & Routing

## 1. Dashboard (The Control Center)
- **Features:** Theme Preset Selector, Background Performance Controller.
- **Background System:** Users can switch between 4 modes (SVG, Canvas, R3F, Gif Mask).
- **Gif Mask Background:** Uses a transparent GIF as a mask layer for unique motion textures.

## 2. Examples Page (The SPA Experience)
- **Structure:** Single Page Application (SPA) feel; renders one "Example Component" at a time.
- **Navigation:** Animated transitions (GSAP) for entering and exiting examples.
- **Container:** Large, centered container with an "Inspect" button in the bottom-right corner.

## 3. The Playground (Full-Screen IDE)
- **Constraint:** Must occupy 100% of the viewport.
- **Integration:** Must receive code snippets from the "Try in Playground" button found in Rosetta Modals.
