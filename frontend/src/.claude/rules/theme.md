# 🎨 Theme Engine (OKLCH Presets)

## Preset Strategy
Discard sliders for a curated Preset Gallery. Each preset defines a Primary, Secondary, and Accent hue.
- **Cultural/Brand:** Apple (Silver/White), McDonald's (Red/Yellow), Chick-fil-A.
- **Sports (NFL/NBA):** Chiefs (Red/Gold), Lakers (Purple/Gold), Celtics (Green/White).
- **Vibe:** Cyberpunk, Neon, Sunset, Sunrise, Space, Deep Sea.
- **Extreme:** Pure White, Pitch Black, Monochrome (with custom tint).

## Generation Logic
- **Grayscale:** Automatically generated by adjusting the Lightness (L) of the Primary OKLCH hue while keeping the Hue (H) constant for brand harmony.
- **Contrast:** Ensure all presets meet WCAG accessibility standards via Zod-validated OKLCH ranges.
